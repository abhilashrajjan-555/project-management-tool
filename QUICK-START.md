# Quick Start Guide

Get the Project Management Tool running in 2 minutes.

## Installation

```bash
# Navigate to project directory
cd project-management-tool

# Install dependencies (if not already done)
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## First Steps

1. **Create a Project**
   - Click "+ New Project" in the sidebar
   - Enter project name and description
   - Click "Create Project"

2. **Add Tasks to Kanban Board**
   - Click "+ Add" in any column (Todo, In Progress, Done)
   - Click "Edit" on the new task
   - Enter title and description
   - Click "Save"

3. **Drag Tasks**
   - Click and drag any task card
   - Drop it in a different column
   - Status updates automatically

4. **Add Todos**
   - Scroll to the Todo List section
   - Type a todo in the input field
   - Press Enter or click "Add"
   - Check boxes to mark complete

## Keyboard Shortcuts

- **Enter** - Submit new todo
- **Escape** - Cancel task editing
